# CekilisHitit

![CekilisHitit Screenshot](../assets/cekilis-hitit.png)

Instagram çekilişlerini ve Gala gecesi etkinliklerini yönetmek için geliştirilmiş masaüstü uygulaması.

## Özellikler
- **Instagram Yorum Çekme**: Belirtilen gönderideki tüm yorumları API üzerinden çeker.
- **Normal Çekiliş Modu**: Asil ve yedek talihlileri belirleme, mükerrer katılım engelleme.
- **Gala Modu**: Görsel efektler ve ses desteğiyle sahne çekilişleri için özel arayüz.
- **Kayıt ve Log**: Yapılan çekiliş sonuçlarını yerel olarak saklar.

## Kurulum
```bash
npm install
npm run dev
```

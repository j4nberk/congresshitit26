# KayitHitit

![KayitHitit Screenshot](../assets/kayit-hitit.png)

Kongre kayıtlarının canlı takibi ve atölye yönetimi için masaüstü paneli.

## Özellikler
- **Canlı Senkronizasyon**: Wordpress sitesinden gelen kayıtları anlık olarak çeker.
- **Atölye Yönetimi**: Hangi öğrencinin hangi atölyeye kayıtlı olduğunu filtreleme ve sıralama.
- **PDF Export**: Listeleri kongre formatında (logolu ve düzenli) PDF olarak çıktı alma.
- **Dashboard**: Katılımcı istatistiklerini izleme.

## Kurulum
```bash
npm install
npm run dev
```

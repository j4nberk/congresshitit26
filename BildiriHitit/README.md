# BildiriHitit

![BildiriHitit Screenshot](../assets/bildiri-hitit.png)

Kongre bildiri yönetim ve puanlama otomasyonu.

## Özellikler
- **Hoca-Bildiri Atama**: Bildirileri ilgili hakemlere/hocalara otomatik veya manuel atama.
- **Puanlama Takibi**: Hocaların verdiği puanları merkezi olarak izleme.
- **E-posta Bildirimi**: Kabul/Red alan öğrencilere toplu ve kişiselleştirilmiş bildirimler.
- **Masaüstü Entegrasyonu**: Electron altyapısı ile hızlı veri işleme.

## Kurulum
```bash
npm install
npm run dev
```

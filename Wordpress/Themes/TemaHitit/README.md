# TemaHitit

![TemaHitit Preview](../../../assets/tema.png)

3. Ulusal Tıp Öğrenci Kongresi için tasarlanmış özel Wordpress teması.

## Özellikler
- **Modern Tasarım**: Kongre konseptine uygun, duyarlı (responsive) tasarım.
- **Gutenberg Desteği**: Özel blok ve pattern yapıları ile düzenlenebilir içerik.
- **Tailwind CSS**: Modern frontend araçları ile hızlı ve optimize edilmiş CSS.
- **Özel Sayfa Şablonları**: Program Atlası, Ekibimiz ve Paketlerimiz gibi özel şablonlar.

## Kurulum
1. `hitit-tema` klasörünü `wp-content/themes/` dizinine kopyalayın.
2. Wordpress panelinden temayı aktifleştirin.

## Geliştirme
```bash
npm install
npm run dev # Tailwind derleme için
```

# CheckPlugin (Registration Check)

Hızlı kayıt kontrolü ve giriş işlemleri için geliştirilen eklenti.

## Özellikler
- **Kayıt Sorgulama**: Katılımcıların kayıt durumlarını hızlıca kontrol eder.
- **reCAPTCHA Desteği**: Form güvenliği için entegre captcha desteği.
- **Ajax Desteği**: Sayfa yenilemeden hızlı sorgulama.

## Kurulum
1. `reg-check-plugin` klasörünü `wp-content/plugins/` dizinine kopyalayın.
2. Wordpress panelinden eklentiyi aktifleştirin.

# KayitPlugin (Kongre Kayıt Plugin)

Özellikle kongre kayıtlarının işlenmesi ve validasyonu için kullanılan eklenti.

## Özellikler
- **Kullanıcı Kayıt Akışı**: Öğrencilerin kongreye kayıt süreçlerini yönetir.
- **Atölye Kayıtları**: Kontenjan takibi yaparak atölye kayıtlarını alır.
- **Ödeme Bilgisi Entegrasyonu**: Manuel veya otomatik ödeme bildirimlerini işler.

## Kurulum
1. `kongre-kayit-plugin` klasörünü `wp-content/plugins/` dizinine kopyalayın.
2. Wordpress panelinden eklentiyi aktifleştirin.

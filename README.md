Bu depo, 3. Ulusal Tıp Öğrenci Kongresi için tarafımca geliştirilen yardımcı masaüstü (Electron/React) uygulamalarını ve Wordpress içeriklerini içermektedir.

## İçerik

### 1. [CekilisHitit](./CekilisHitit)
Instagram yorumlarını çekmek ve çekiliş (normal & gala modu) yapmak için kullanılan uygulama.

### 2. [BildiriHitit](./BildiriHitit)
Kongre bildiri yönetim sistemi. Hocaların puanlaması ve öğrencilere otomatik bilgilendirme gönderimi sağlar.

### 3. [MailHitit](./MailHitit)
Excel listeleri üzerinden kişiselleştirilmiş toplu e-posta gönderim aracı.

### 4. [KayitHitit](./KayitHitit)
Kongre kayıtlarını  canlı olarak izleme, sıralama ve atölye listelerini PDF olarak dışa aktarma aracı. Sadece kongre-kayit plugini ile entegre çalışır.

### 5. [Wordpress](./Wordpress)
Kongre [sitesinin](https://congresshitit.com.tr/) Wordpress eklentileri ve temalarını içerir.

## Kurulum ve Çalıştırma

Her uygulama kendi dizini içinde bağımsız bir React/Electron projesidir. Çalıştırmak için ilgili klasöre gidip aşağıdaki komutları kullanabilirsiniz:

```bash
npm install
npm run dev
npm run build
```

## Pluginler

Bu uygulamaların bağlı olduğu ve birlikte çalıştığı Wordpress Plugin'leri ve Temaları için [Wordpress](./Wordpress) klasörüne bakabilirsiniz.
